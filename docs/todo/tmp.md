# UI レイアウト方針メモ

## メインエリア（デュアルペイン）

ヘッダーの下にある主要な表示領域。

- 左：Google Maps（約 60%）
- 右：検索結果リスト（約 40%）

このレイアウトは、**地図上で位置関係を把握しながら、詳細情報を確認する**というユーザーニーズに応える。

---

## Google Maps（左ペイン）

- 画面左側、全体の約 60%を占める領域。
- 検索結果の店舗がマーカー表示。
- 店舗名・塩分量などの情報を簡潔に表示。

→ **直感的な地図上での店舗の位置把握と視覚的な比較**が可能。

---

## 店舗リスト（右ペイン）

- 画面右側、全体の約 40%を占める領域。
- 各店舗は横長の「カード」形式で表示。
- カードには店舗名、住所、塩分量、現在地からの距離など主要情報をまとめて表示。

→ **店舗ごとの詳細な情報を一覧形式で確認できる**。
